# Heros League - Site Vitrine

## Description

Ce projet consiste en la création d'un site vitrine pour travailler les langages **HTML** et **CSS**. Le site met en avant des services fictifs et inclut des éléments interactifs et visuels.

## Fonctionnalités

-   **Structure HTML** : Mise en place d'une structure sémantique.
-   **Styles CSS** : Utilisation de styles modernes avec des variables CSS pour les couleurs et les bordures.
-   **Responsive Design** :
    -   **880px** : Adaptation pour les tablettes et écrans moyens.
    -   **400px** : Optimisation pour les écrans mobiles.

## Technologies utilisées

-   **HTML5**
-   **CSS3**

## Instructions

1. Clonez le dépôt ou téléchargez les fichiers.
2. Ouvrez le fichier `index.html` dans un navigateur pour visualiser le site.
